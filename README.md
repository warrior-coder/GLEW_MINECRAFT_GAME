# OPENGL MINECRAFT GAME

Voxel Engine on C++ with modern OpenGL. An attempt to create a game somehow similar to the famous Minecraft.

Using libraries:
* GLEW;
* GLFW;
* GLM;
* LIBPNG;
* ZLIB.
